Component Details

This is a demo of the ResourceCard React component. The component displays a resource card with:

  • Title
  • Description
  • Image
  • Download button
  • Optional video guide link

The component features:

  • Hover effects (shadow and scale)
  • Responsive design
  • Consistent styling with the website theme

To use this component in a React project, you would need to install Node.js and npm, then create a React application.
